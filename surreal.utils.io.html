class surreal.utils.io.logger.LoggerFormatAdapter(logger)[source]¶

Bases: object

logger = LoggerFormatAdapter(logger) fully supports positional/keyword new formatter string syntax

Example:

::
logger.info(“Float {1:>7.3f} and int {0} with {banana} and {apple:.6f}”,
66, 3.141592, apple=7**.5, banana=’str’)

# if the first arg isn’t a format string, treat as if print statement # you can specify a sep= arg under this case. sep defaults to space logger.info({3:’apple’, 4:’banana’}, 4.5, ‘asdf’) logger.info(‘I am not a format string’, 66, {‘key’:’value’}, sep=’, ‘)

Custom verbosity level. The higher the more strings printed. log.info1(), .info2() ... info5(). corresponding levels: INFO1 to INFO5
